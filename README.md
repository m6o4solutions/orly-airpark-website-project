## Introduction

This is Orly Airpark, a charming airfield located in Kenya's Rift Valley. Orly Airpark is
a hub for adventurers and thrill-seekers looking to explore the Rift Valley and its
surrounding areas from above. With a well-maintained runway and modern facilities, Orly
Airpark offers a unique flying experience for both experienced pilots and first-time
flyers.

## Features

- **Dynamic Content:** Engage visitors with dynamic and personalized content.
- **Interactive UI/UX:** Provide an immersive user experience with smooth navigation.
- **Service Showcase:** Highlight services available at Orly Airpark.
- **Contact Form:** Enable clients to reach out easily for inquiries and collaborations.

## Tech Stack

- **Frontend:** Next.js version 16, TypeScript, Tailwind CSS and Shadcn UI
- **Backend:** Payload CMS with MongoDB database
- **Deployment:** Vercel

## Getting Started

1. Clone the repository.
2. Install dependencies using `pnpm install`.
3. Run the development server with `pnpm dev`.
